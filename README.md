# Salary-Analysis
Interactive Excel dashboard analyzing data science salaries, skills, and job market trends using Power Query, Power Pivot, and DAX.

# 📊 Salary Analysis Dashboard (Excel)

## Project Overview

This project is part of my Excel for Data Analytics learning journey. The goal was to analyze salary data from data-related jobs and understand how salaries vary based on job role, skills, and location. I used advanced Excel features to clean the data, build a data model, and create interactive dashboards.

---

## Questions Explored

- Does having more skills lead to a higher salary?
- How do salaries differ across countries?
- Which skills are most commonly required for data jobs?
- Which skills are associated with the highest salaries?

---

## Tools & Skills Used

- Microsoft Excel
- Power Query
- Power Pivot
- Pivot Tables
- Pivot Charts
- DAX Measures
- Data Model
- Slicers

---

## What I Did

- Cleaned and transformed the raw dataset using Power Query.
- Built relationships between multiple tables using Power Pivot.
- Created DAX measures to calculate median salaries.
- Used Pivot Tables and Pivot Charts to analyze the data.
- Designed an interactive dashboard to explore salary trends.

---

## Key Findings

- Jobs requiring more technical skills generally offer higher salaries.
- Senior Data Engineers and Data Scientists have some of the highest median salaries.
- Python and SQL are among the most in-demand skills.
- Cloud technologies like AWS and Azure are becoming increasingly important.
- Salary levels vary across different regions and countries.

---

## What I Learned

This project helped me improve my Excel skills by working with real-world data. I gained practical experience in data cleaning, data modeling, DAX calculations, dashboard creation, and data visualization. It also strengthened my understanding of how to present business insights using Excel.
